# 💼 AI Market Job Analyzer

An easy-to-use Python platform that analyzes the tech job market. It takes job descriptions or resumes, processes them using advanced AI/LLMs, and helps you extract key skills, match jobs, and make data-driven career choices.

---

## ✨ Features

*   **🔍 Smart Job Analysis:** Pulls and structures raw tech job listings automatically.
*   **🧠 AI-Powered Extraction:** Uses Large Language Models (LLMs) to scan text and pull exact tech stacks, certifications, and requirements.
*   **📈 Career Insights:** Matches user profiles against job descriptions to show skill gaps and compatibility scores.
*   **📊 Interactive Dashboard:** A clean user interface built with Streamlit for quick, real-time analysis.

---

## 🛠️ Built With

*   **Python**
*   **Streamlit** (For the web dashboard)
*   **LLM Integration** (Groq / Gemini / OpenAI API for advanced text analysis)
*   **Pandas** (For data structuring)
